Impor Data Siswa Melalui File CSV

Anda dapat mengimpor banyak data siswa sekaligus menggunakan format standar CSV (Comma Separated Values) yang kompatibel dengan Microsoft Excel.

Panduan Pengisian:

  • Unduh file template di samping kanan.
  • Buka file di Microsoft Excel, Google Sheets, atau text editor lainnya.
  • Isi kolom nama, username, password dengan benar.
  • Simpan kembali dalam format CSV (Comma Delimited).
  • Pilih dan upload file tersebut pada formulir di bawah.
